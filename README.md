# HYE-backend

Backend of Have You Eaten?

## Requirements

Python

- install 3.12.4 venv

```
source env/bin/activate
```

Pip

```
pip install -r requirements.txt
```

OR if you prefer to do it manually

```
pip install fastapi \
            uvicorn \
            bcrypt \
            pydantic \
            secure-smtplib \
            python-jose
```
